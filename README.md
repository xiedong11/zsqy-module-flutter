# 掌上教务
   
### 帮跑腿 flutter端

### 业务说明
  
  >该项目为纯flutter端项目，采用aar方式寄生在原生APP中，作为掌上教务中的一个独立模块
  在业务逻辑上做到与原生APP完全隔离，Flutter端开发者，可完全不用关注原生端的业务模块
  两端开发彼此业务隔离，缩小了对开发人员的业务能力要求，同时也可以上flutter端开发者
  快速上手到开发中去
